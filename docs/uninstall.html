# Uninstall

The Monzo CLI consists of two parts which you will want to remove when uninstalling the application. These are the binary itself along with application data (which stores your encrypted credentials and personal configurations).

# Removing the Application

Removing the binary depends on which method you used to initially install the application:

# Homebrew
$ brew uninstall mzo
$ brew untap jamesstidard/formulas

# Removing Application Data

Application data is stored in a hidden folder in your home directory named .mzo. This can be removed from your terminal with:

$ rm -rf ~/.mzo
